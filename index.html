Privacy Policy

Fonts in Marathi – मराठी Font Style

This Privacy Policy explains how our Chrome extension handles user data.

Good News: This extension does NOT collect or store any personal data.

1. Information We Collect

We do not collect, store, or share any personal or sensitive user data.

  • No personal information (name, email, etc.)
  • No browsing history
  • No location data
  • No user activity tracking

2. How the Extension Works

All features of this extension work locally in your browser:

  • Text input and font generation happen inside your device
  • No data is sent to external servers
  • Font previews and downloads are processed locally

3. Permissions

If storage permission is used, it is only for saving user preferences such as:

  • Recently entered text
  • Selected font styles
  • UI settings (like theme)

All stored data remains on your device and is never shared.

4. Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

5. Data Security

Since no data is collected or transmitted, your privacy is fully protected.

We may update this Privacy Policy in the future. Any changes will be posted on this page.
